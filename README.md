En este repositorio, se encuentra armada una base de datos sencilla armada con PostgreSQL para la compleción del módulo 5 de bases de datos relacionales, la cual tiene el funcionamiento interno de una clínica veterinaria.
Los archivos acá contenidos son los siguientes:

- Archivo .sql con el desarrollo y la creación de la base de datos junto a sus registros y las operaciones solicitadas.
- Presentación en PowerPoint analizando el caso, mostrando la estructura de la base de datos, y explicando las razones de ser de la sintaxis de las consultas armadas para traer los datos.
- Archivo README.md, el cual estás leyendo ahora mismo

El ejercicio impuesto ha sido ideado por Kibernum y su bootcamp de JavaScript. La solución fue desarrolada por BifoldTide utilizando solamente PostgreSQL.
